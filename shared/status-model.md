# Status Model

## Framework Status

Shared framework lifecycle vocabulary:

- `intake`
- `analyzed`
- `experimenting`
- `evaluated`
- `decided`

## Runtime Status

Shared runtime delivery vocabulary:

- `none`
- `planned`
- `implementing`
- `callable`
- `parked`
- `removed`

## Interpretation

- Framework can be `decided` without runtime delivery.
- Runtime is complete only at `callable` with proof.
- Architecture outcomes may end without any runtime status change.
