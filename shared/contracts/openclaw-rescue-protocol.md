# OpenClaw Rescue Protocol

Last updated: 2026-03-22
Status: reference-only

Directive Workspace does not own the OpenClaw rescue protocol.

Canonical external rescue protocol:
- `C:\Users\User\.openclaw\workspace\rescue-openclaw\RESCUE_PROTOCOL.md`

Canonical external rescue surface:
- `C:\Users\User\.openclaw\workspace\rescue-openclaw\README.md`

Directive Workspace may reference this protocol only when a Directive-owned document needs to point at the external OpenClaw recovery surface.
It should not redefine the rescue workflow, handoff model, or ownership for Rescue OpenClaw.
