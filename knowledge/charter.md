# Workspace Charter

## Mission

Directive Workspace is an objective-driven system that continuously identifies mission-relevant capability gaps, extracts useful mechanisms from existing and emerging breakthroughs, proves them safely, and upgrades the architecture/runtime/agent stack in service of the active mission.

Core purpose: convert breakthroughs into usefulness, where usefulness is defined by the active mission.

The correct hierarchy is:
- **Directive Workspace**
  - whole product
- **Engine**
  - shared core machinery
- **Discovery / Runtime / Architecture**
  - the three operating lanes of the Engine

Mission Control is the active runtime host. OpenClaw is the persistent orchestration layer.

## Success Metrics
- `decision_lead_time_hours` is stable and trending down.
- `adopt_to_callable_lead_time_hours` is measured and improving.
- Nightly `check:ops-stack` remains green.
- Capability decisions are evidence-backed (not intuition-only).
- Runtime integrations have clear rollback paths.

## Constraints
- No blind external repository adoption into runtime.
- No runtime integration without promotion contract and proof artifact.
- Keep scope bounded by mission, proof, and rollback; avoid broad autonomy expansion without explicit usefulness.
- Keep changes modular so runtime and framework can evolve independently.

## Current Priorities
- Priority 1: strengthen Discovery as the real mission-aware front door and reduce the intake-to-routing coverage gap.
- Priority 2: keep Runtime focused on bounded runtime operationalization and behavior-preserving transformation with explicit proof.
- Priority 3: maintain operational reliability while improving evaluation quality through dimensional evaluators, not just binary gates.
- Priority 4: resolve or retire remaining agent-lab-era bridge assumptions so ownership stays clean.

Note: Architecture cycle completion is 100% for the current adopted set (see architecture-completion-rubric.md). Current priorities reflect the shift from structural buildout to operational proof.
