# Runtime Registry

This folder holds accepted Runtime runtime entries.

Use:
- `shared/templates/registry-entry.md`

An entry belongs here only after:
- a promotion record exists
- the required host gates have passed
- proof is linked
- runtime status is explicit
- new accepted entries pass `runtime_registry_acceptance_gate.v1`, including
  a host callable adapter report and callable execution evidence when execution
  is claimed

This is the product-owned view of accepted callable/runtime capability, even when execution is hosted elsewhere.
